# PF-Project-Wordle
Wordle is a word guessing game inspired from the original Wordle by nytimes . 

This project is a console-based Wordle game developed as part of my Programming Fundamentals (PF) course. The game is inspired by the popular Wordle puzzle, where the player has a limited number of attempts to guess a hidden word. After each guess, feedback is given to help the player reach the correct answer.

The main goal of this project is to demonstrate a practical understanding of basic programming concepts such as variables, loops, conditions, functions, arrays/strings, and user input/output.
